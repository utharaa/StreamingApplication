# Streaming application

streaming application which reads data from realtime bitcoin transactions

# Installation

execute producer.py
execute consumer.py
and run the run.py

# API details

transactions_count_per_minute

```
    http://127.0.0.1:5000/transactions_count_per_minute
    
    Response
    {"20:03": 251, "20:02": 254}
 ```
